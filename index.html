Conceptos Básicos de JavaScript

document.write

La función document.write() se utiliza para escribir contenido en el documento HTML en tiempo de carga.

alert

El método alert() muestra un cuadro de diálogo con un mensaje en el navegador.

prompt

La función prompt() muestra un cuadro de diálogo con un mensaje y permite al usuario ingresar información.

Variables: let y const

Las variables en JavaScript se declaran con let o const. let permite reasignación, mientras que const declara una constante.

Operadores Lógicos

Los operadores lógicos como && (AND), || (OR) y ! (NOT) se utilizan para realizar operaciones lógicas entre valores booleanos.
